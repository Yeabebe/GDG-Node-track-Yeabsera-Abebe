# GDG Node Task 2

This project is part of the GDG Node track. It demonstrates the use of Node.js for building a simple application.

## Features

- Basic application structure
- Example of using Node.js

## Getting Started

### Prerequisites

- Node.js installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd task2
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Usage

To run the application, use:

```bash
node app.js
```
